阿里云CLI
阿里云命令行工具 CLI（Alibaba Cloud CLI）是基于阿里云开放 API 建立的管理工具。借助此工具，您可以通过调用阿里云开放 API 来管理阿里云产品。该命令行工具与阿里云开放 API 一一对应，灵活性高且易于扩展。您可基于该命令行工具对阿里云原生 API 进行封装，扩展出您想要的功能。
