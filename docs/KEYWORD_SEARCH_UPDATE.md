## 🎮 使用方法

1. **选择搜索模式**: 点击 "🔍 智能行业搜索"
2. **输入关键词**: 如 "Python编程"、"Docker容器"
3. **设置参数**: 
   - 递归深度: 2层 (推荐)
   - 每层页数: 20页 (推荐)
4. **开始搜索**: 系统会在10个高质量网站中搜索相关内容

## 🔧 扩展配置

### **添加新网站**:
编辑 `src/config/top20_sites.py`，在相应分类中添加:
```python
{
    "name": "新网站名称",
    "url": "https://example.com/",
    "difficulty": 2,  # 1-5星难度
    "description": "网站描述"
}
```

### **自定义搜索列表**:
修改 `DEFAULT_SEARCH_SITES` 和 `DEFAULT_SITE_NAMES` 列表。
