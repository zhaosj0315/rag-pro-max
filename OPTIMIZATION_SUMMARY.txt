================================================================================
  快速升级步骤
================================================================================

1. 备份数据
   cp -r vector_db_storage vector_db_storage.backup
   cp -r chat_histories chat_histories.backup

2. 更新代码
   git pull origin main

3. 安装依赖
   pip install -r requirements.txt

4. 清理缓存
   rm -rf src/__pycache__
   rm -rf .streamlit/cache

5. 重启应用
   ./start.sh
